# 清泽心雨设备管理
* 清泽心雨网站下设设备管理功能
# 功能
* 实现手机，电脑对清泽心雨特种设备对借还
* 清泽心雨后台程序对设备借还情况的可视化查询
# 构建特色
* 使用易班开发者接口，使用易班账号登陆，减少注册用户环节
* 本校用户登陆后即可对设备进行借还
* 后台记录各个设备情况以及各用户借还情况
# 构建任务
* 易班账号登陆页面
* 选择设备借还页面
 > 查看当前个人借还信息
 > 借设备入口
 > 还设备入口
* 个人借还信息页面
 > 个人信息
 > 设备信息
* 接设备入口
 > 单反，单反卡，电池，大摄像机，三脚架，马甲
 > 剩余情况
* 还设备入口
 > 已借设备
 > 一键偿还
 > 单个偿还


* 后台页面
 >设备总览
 >借出总数
 >偿还总数
 >各设备占重比例
 >选择各个设备
 >查询某人借还信息
  >>总次数
  >>占重比例
